Google SEO手段及操作方案

什么是Google优化

  1. 利用搜索引擎算法把网站排到SERP首页
  2. 工作原理
    1. 爬虫抓取、提交sitemap
    2. 收录
    3. 排名
  3. 白帽SEO
    1. 完全以用户体验为中心，遵循Google价值观
    2. 正规优化手段，排名长久
    3. 缺点：优化时间长
  4. 黑帽SEO
    1. 搜索引擎算法漏洞
    2. 缺点：死得快

站内优化

  1. 技术优化
    1. 打开速度
      1. PC打开速度
      2. 移动端打开速度
    2. HTTPS安全锁
    3. 网站URL层级架构
      1. 扁平化处理
      2. 不超过3级
    4. 页面锚文本内链建设
      1. 对应关键词链接到对应页面
      2. 避免过度使用锚文本
    5. TDK设置
      1. title含关键词
      2. description含业务关键词
      3. 各个页面的关键词不重复堆砌
  2. 内容营销
    1. What：提供能解决用户问题，高质量的内容，干活满满
    2. Why：用户喜欢高质量的内容，Google喜欢用户喜欢的
    3. How：用户痛点调研
      1. 关键词调研
        1. Google Keyword Planner
        2. SEMRUSH
        3. Ubersuggest
        4. Google Suggest
      2. Amazon/速卖通差评
      3. 论坛调研
      4. 问答网站Quora/Reddit
      5. Answer The Public
      6. Twitter/Youtube客户对产品的评价
  3. 保持更新/咖啡因更新
    1. 咖啡因系统把网络分成小区块，持续更新索引库，发现新页面，或老页面上的新内容，直接进入索引库

站外优化/外链建设

  1. What：互联网的投票，谁的票数多谁就排在前面
    1. Dofollow对排名有用
    2. Unfollow对排名没用（间接引流）
  2. Why：Google排名最重要的因素
  3. How：
    1. 寻找站长
      1. 行业相关度高
      2. Best Keyword Blog
      3. Top Keyword Blog
      4. inurl:keyword blog
      5. intitle:keyword blog
    2. Outreach站长
      1. 免费
        1. 先成为站长的粉丝，保持互动
        2. 主动发邮件分享自己的文章并适当的体积能否给链接
      2. 付费
        1. 产品测评。给出测评的角度及关键词的锚文本链接
        2. 付费写文章。不少于1000字，但规定内容的方向和框架
    3. 负面评价处理

Tellus 具体操作方案

  1. 站内：整站彻查
    1. 整站URL结构清检
    2. TDK关键词规划与重设置
    3. 内链/锚链接扫描与设置
    4. 死链清检
    5. HTTPS安全锁清检
    6. 页面打开速度清检
    7. sitemap设置与提交
  2. 站内：追加内容更新版块
    1. 内容更新是目前网站所没有的版块，我建议重点操作此模块。坚持长期优质的内容更新可使排名飞快上升且稳固在前端
    2. wordpress简易搭建
    3. 内容持续更新
      1. 原创
      2. 伪原创
      3. 搬运
    4. sitemap持续追加内容页面并提交
  3. 站外
    1. 外链
      1. 交换友情链接
      2. 寻找自由站长、博主
    2. 社区维护/站外原创
      1. 相关问答社群及论坛
      2. 其他
    3. 付费方案（可选）
      1. 测评
      2. 文章
    4. 定期检索负面评价并处理
  4. 品牌打造
    1. 通过品牌带动搜索实现外部点击增长。 但要留住用户，还需上述内容模块的支撑
    2. 条件允许的话，做原创短视频输出产品核心价值，上传Youtube并分享 视频营销如今越来越普及，随着5G到来，视频甚至可能应用到Web页面
    3. 热门手机App社群如Facebook、Twitter、Instagram等社群建立粉丝群 做活微营销与粉丝经济
